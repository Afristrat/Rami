## VECTEUR 1 — PUBLISHING & PLATEFORMES

### 1.1 Formats de contenu manquants (CRITIQUE)
| Format | Plateformes | Status | Impact Business |
|--------|------------|--------|----------------|
| **Instagram Reels** | Instagram | ❌ MISSING | 91% des agences demandent |
| **Instagram Stories** | Instagram | ❌ MISSING | Format #1 engagement |
| **YouTube Shorts** | YouTube | ❌ MISSING | Croissance explosive |
| **TikTok Videos** | TikTok | ❌ MISSING | Gen Z + viral |
| **LinkedIn Carousels** (PDF) | LinkedIn | ❌ MISSING | Top format B2B 2025 |
| **LinkedIn Newsletters** | LinkedIn | ❌ MISSING | Audience captive |
| **X/Twitter Threads** | X | ❌ MISSING | Format viral X |
| **Facebook Reels** | Facebook | ❌ MISSING | Reach organique |
| **Pinterest Idea Pins** | Pinterest | ❌ MISSING | Format long-form Pinterest |
| **YouTube Community Posts** | YouTube | ❌ MISSING | Engagement abonnés |
| **WhatsApp Status** | WhatsApp | ❌ MISSING | Marché MENA |
| **WhatsApp Broadcast** | WhatsApp | ❌ MISSING | Lead nurturing |

**Verdict :** RAMI publie uniquement du contenu "post image + texte". C'est 2020. En 2026, le video-first est non-négociable.

### 1.2 Plateformes manquantes (Phase 2 non démarrée)
- **TikTok Business** — 0 implémentation
- **YouTube Channel** — dans le design Phase 1 mais code Phase 2
- **WhatsApp Business API** — critique pour le marché MENA/Maroc
- **Snapchat** — jeune audience MENA
- **Telegram** — canaux broadcast, très utilisé Maroc/Afrique
- **Google Business Profile** — souvent oublié, ESSENTIEL pour SEO local
- **Threads (Meta)** — nouveau mais croissance rapide
- **Bluesky** — émergent pour niches tech/créateurs

### 1.3 Features publishing avancées manquantes
- **First Comment Auto-post** — poster les hashtags en 1er commentaire (Instagram best practice)
- **Watermarking auto** — logo de l'agence/marque sur chaque visuel
- **UTM builder intégré** — générer les UTM pour chaque post publié
- **Link tracking** — clicks par post via URL shortener intégré (type bit.ly)
- **Cross-posting intelligent** — adapter automatiquement le format selon la plateforme (ratio image, longueur texte, hashtags)
- **Géo-targeting** — publier pour une audience géographique (Facebook/Instagram)
- **Audience targeting** — cibler une audience spécifique lors de la publication

## VECTEUR 2 — CONTENU & IA GÉNÉRATIVE

### 2.1 Types de contenu IA manquants
- **Génération vidéo** — Runway ML / Kling AI / Pika Labs pour Reels/Shorts
- **Avatar IA** — spokesperson virtuel pour vidéos (HeyGen/D-ID)
- **Text-to-Speech** — voix off pour vidéos (ElevenLabs)
- **Image-to-Video** — animation de visuels fixes en Reels
- **Carousel auto-builder** — PDF→LinkedIn carousel auto-formaté
- **Caption auto-transcription** — sous-titres auto sur vidéos
- **Hashtag research IA** — suggérer les meilleurs hashtags par niche/région en temps réel
- **Trending topics integration** — alertes sur les sujets trending par plateforme
- **Content recycling IA** — identifier les top posts et les re-proposer à republier (evergreen)
- **A/B testing contenu** — tester 2 versions d'un post, analyser laquelle performe mieux

### 2.2 Qualité génération texte
- **Tone of Voice checker** — vérifier que le texte généré respecte vraiment le Brand DNA éditorial
- **Plagiarism check** — s'assurer que le texte n'est pas copié
- **SEO scoring** — score SEO pour les descriptions YouTube, légendes Instagram
- **Readability score** — Flesch-Kincaid adapté au français/arabe
- **Multi-langue** — Génération en arabe (darija/fusha), anglais, français simultanément = CRITIQUE pour Maroc
- **RTL support** — texte arabe de droite à gauche dans les visuels générés

### 2.3 Visual Engine gaps
- **Brand Kit enforcement** — vérification automatique que les visuels respectent la charte (police, couleurs exactes)
- **Template library** — 100+ templates pré-faits par secteur/occasion (Ramadan, fêtes, promotions)
- **Resize intelligent** — 1 visuel → 12 formats plateformes auto (1:1, 9:16, 16:9, 4:5)
- **Remove background** — suppression fond auto sur photos produits
- **Face swap / product placement** — IA pour intégrer produits dans scènes
- **Text overlay IA** — ajouter texte sur image avec typographie Brand DNA automatique
- **Color grading** — adapter les couleurs d'une photo existante à la palette Brand DNA
- **Mockup generator** — mettre le visuel sur des mockups (téléphone, billboard, t-shirt)

## VECTEUR 3 — ANALYTICS & INTELLIGENCE

### 3.1 Analytics manquantes (CRITIQUE)
- **ROI tracking** — lier les posts à des conversions (ventes, leads, clics)
- **Competitor analysis** — suivre les comptes concurrents (likes, followers, engagement)
- **Social listening** — mentions de marque sur toutes les plateformes
- **Sentiment analysis** — tonalité des commentaires (positif/négatif/neutre)
- **Audience demographics** — âge, genre, localisation, heures d'activité
- **Best time to post IA** — recommander le meilleur créneau par plateforme et audience
- **Content attribution** — quel contenu a généré des conversions
- **Hashtag analytics** — performance des hashtags utilisés
- **Story analytics** — vues, sorties, tap-forwards pour Stories
- **Video analytics** — durée de visionnage, drop-off rate
- **Benchmark sectoriel** — comparer les KPIs vs concurrents du même secteur
- **Rapport automatique hebdomadaire** — PDF envoyé au client chaque lundi matin

### 3.2 Intelligence Brand DNA
- **Brand DNA score historique** — graphe 30j visible dans le design mais non implémenté
- **Alertes cohérence** — notifier si un post dévie du Brand DNA
- **Cross-client insights** — identifier les couleurs/styles qui performent par secteur (anonymisé)
- **Causse matrix évolution** — tracker comment les émotions cibles évoluent dans le temps

## VECTEUR 4 — COLLABORATION & WORKFLOW AGENCE

### 4.1 Team & Approval workflow (CRITIQUE pour agences)
- **Rôles d'équipe** — Admin, Manager, Créatif, Client (lecture seule) : MISSING
- **Approval workflow** — soumission → revue manager → approbation client → publication
- **Commentaires sur posts** — thread de commentaires sur chaque post avant publication
- **Versioning de contenu** — historique des révisions d'un post
- **Client portal dédié** — URL unique pour que le client valide son contenu sans accès au dashboard complet
- **@mentions dans commentaires** — notifier un collègue
- **Task assignment** — assigner la création d'un post à un membre d'équipe
- **Deadlines sur contenus** — date limite de livraison visible

### 4.2 Multi-projet & organisation
- **Projets** (visible dans le design Bibliothèque mais non implémenté) — grouper les contenus par campagne/projet
- **Campagnes** — organiser les posts par campagne marketing avec dates début/fin
- **Labels/Tags personnalisés** — taguer les contenus (Ramadan, Promo, Brand Awareness...)
- **Content calendar partageable** — URL publique du calendrier pour le client
- **Bulk scheduling** — importer un CSV avec 50 posts d'un coup

### 4.3 Onboarding client
- **Client intake form** — formulaire que le client remplit pour démarrer (Brand DNA auto-populé)
- **Brand DNA import** — importer depuis un fichier de charte graphique existant
- **Competitor import** — analyser automatiquement les comptes RS concurrents pour calibrer le Brand DNA

## VECTEUR 9 — UX & PRODUCT QUALITY

### 9.1 UX manquante
- **Onboarding interactif** — checklist "démarrez ici" avec progression visible (style Linear)
- **Tooltips contextuels** — explications inline sur les features (Brand DNA score, Causse, etc.)
- **Empty states** — messages d'encouragement et CTA quand les données sont vides
- **Keyboard shortcuts** — navigation rapide pour power users
- **Dark/Light mode toggle** — (actuellement tout dark)
- **Mobile responsive** — le dashboard est-il utilisable sur mobile ?
- **PWA** — Progressive Web App pour accès mobile sans app store
- **Offline mode** — brouillons accessibles offline

### 9.2 Performance & fiabilité
- **Image lazy loading** — la bibliothèque avec 100+ assets doit loader proprement
- **Infinite scroll vs pagination** — UX bibliothèque
- **Optimistic UI updates** — actions qui s'affichent immédiatement sans attendre le serveur
- **Error recovery** — si une publication échoue, l'UI guide vers la solution
- **Retry UI** — bouton "Réessayer" visible en cas d'échec publication

## VECTEUR 10 — MARCHÉ MENA/MAROC SPÉCIFIQUE

### 10.1 Features critiques pour le marché cible
- **Support arabe (darija + fusha)** — génération de contenu en arabe
- **RTL interface** — interface de droite à gauche pour utilisateurs arabophones
- **Calendrier islmique** — dates Ramadan, Aïd, etc. dans le calendrier de publication
- **Codes couleur culturels** — la matrice Causse adaptée aux codes culturels marocains/africains est présente mais est-elle vraiment fine ?
- **WhatsApp Business** — canal #1 au Maroc pour les entreprises, critique
- **Dirham (MAD) support** — dans la facturation (pas juste USD/EUR)
- **CNDP compliance affichée** — badge/mention visible dans l'app pour rassurer les agences marocaines
- **Contenu multilingue** — FR + AR + EN dans le même post (code-switching fréquent au Maroc)
